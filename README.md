# Sistema de Gestión de Pacientes - Java

Aplicación de consola desarrollada en Java para el registro y gestión de pacientes, enfocada en el manejo de datos, validación de información y lógica de negocio.

## Descripción

Este sistema permite administrar información básica de pacientes mediante estructuras dinámicas, implementando validaciones y cálculos automáticos.

El proyecto está orientado a reforzar conceptos fundamentales de programación como:
- Manejo de datos
- Lógica de negocio
- Validación de entradas
- Estructuras dinámicas

## Funcionalidades

- Registro de pacientes
- Validación de datos de entrada
- Cálculo automático de edad
- Consulta de información de pacientes
- Manejo de excepciones personalizadas
- Almacenamiento temporal en estructuras dinámicas (ArrayList)

## Tecnologías

- Java
- Programación orientada a objetos
- Estructuras dinámicas (ArrayList)
- Manejo de excepciones
